Example to execute in command Line in silent mode

Windows:
"C:\Program Files (x86)\Adobe\Adobe ExtendScript Toolkit CC\ExtendScript Toolkit.exe" -cmd "Execute ExtendedScript Toolkit Example.jsx"

MAC OS X:
"ExtendScript Toolkit.app/Contents/MacOS/ExtendScript Toolkit" \
-cmd "Execute ExtendedScript Toolkit Example.jsx"
